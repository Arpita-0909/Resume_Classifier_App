# Resume_Classifier_App

## Project Overview
The **Resume Classifier App** is a machine learning-based web application that automatically classifies resumes into different job categories using **Natural Language Processing (NLP)** techniques.

This helps recruiters quickly screen and categorize resumes efficiently.

---

##  Features
- 📂 Upload resume (PDF format)
- 🧹 Automatic text extraction from resumes
- 🧠 NLP-based text preprocessing
- 🤖 Machine learning model for classification
- 📊 Displays predicted job category
- 📈 Shows similarity scores (if implemented)
- ⚡ Fast and user-friendly interface using Streamlit

## 🛠️ Tech Stack
- Python 
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- PDFMiner (for text extraction)
- TF-IDF Vectorizer
